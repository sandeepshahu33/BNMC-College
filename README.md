# 🎓 BNMC College Website

A modern, fully responsive multi-page college website developed using **HTML**, **CSS**, and **JavaScript**. This project was created to simulate a professional educational institution's website, focusing on an engaging user experience, clean design, and smooth navigation across all devices.

The website features multiple dedicated pages covering various aspects of a college, including academic information, administration, admissions, extracurricular activities, and institutional resources. Special attention has been given to responsive layouts, interactive animations, and modern UI design to deliver a professional browsing experience.

## ✨ Key Features

- 📱 Fully Responsive Design
- 🎨 Modern and Clean User Interface
- ⚡ Smooth Scroll & Animation Effects
- 🖼️ Interactive Image Sliders
- 📄 Multi-Page Website Architecture
- 🚀 Fast Loading Performance
- 🧭 User-Friendly Navigation

## 🎯 Project Objectives

The primary objective of this project was to design and develop a professional college website that demonstrates modern front-end development practices. During development, the focus was placed on creating a responsive layout, implementing smooth animations, improving user experience, and organizing content in a structured and accessible manner.
